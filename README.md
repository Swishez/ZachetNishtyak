Для работы с файлом нужны следующие требования: Python 3.13.3, python-docx, flask

Чтобы скачать "python-docx" пишите команду: pip install python-docx (Не видоизменяйте команду иначе не будет работать).
Чтобы скачать "flask" пишите команду: pip install flask (Не видоизменяйте команду иначе не будет работать).
